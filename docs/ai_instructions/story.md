# Story Mode AI Instructions

## Character Card Format

**Required Fields:**
- `name`: Character's full name/title
- `description`: Physical appearance (50-100 words) - striking features, age/build, distinguishing marks, clothing, mannerisms
- `personality`: Core traits (50-100 words) - temperament, 3-5 key traits, contradictions, motivations/fears, behavioral patterns
- `example_dialogue`: 3 varied speech examples showing unique voice/tone

## Lorebook Entry Format

**Required Fields:**
- `keys`: Array of 3-5 activation keywords (case-insensitive, natural terms players would use)
- `content`: World/lore info (100-200 words) - what/where/who, specifics, history (when/by whom/why), current state, politics/power dynamics, conflicts/opportunities

## Critical Rules
- **Description**: Physical details + body language only (not personality)
- **Personality**: Traits + depth + contradictions only (not appearance)
- **Dialogue**: Unique voice patterns (not generic speech)
- **Keys**: Natural player terms (not jargon)
- **Content**: Specific actionable details (not vague lore)
